# About Romanization

Shokuhō has a vast number of romanization errors. These consist of simply the author's careless mistakes and the absence of the general romanization system in Japan. Currently, the romanization of Japanese is not standardized. Although there are two romanization styles, Hepburn style and Kunrei style, both are incomplete in terms of both Japanese phonology and orthography.

In this mod, I used my own variant of the BGN/PCGN Japanese Romanization 2017 Style (so-called Modified Hepburn style). The BGN/PCGN style is well-defined for pronouncing the Japanese language, while it is not transcribed from Kana letters. For example, the letter "ō" isn't distinct between "おお" and "おう" in Kana letters, both of which are pronounced [oː]. In my style, I introduced successive vowels "ou," as the transcription from "おう," which has identical pronunciation with "おお" but should be discriminated from the other, in the view of orthography. The only exception is Shokuhō (しょくほう), the title of this mod.  This is the only change from the original NGN/PCGN style. The transcription is based on the modern orthography, not the old orthography at that period.

ご存知のように, 現在, 日本語のローマ字表記は公式には統一されておらず, なおかつ不完全です. 現在はヘプバーン (ヘボン) 式と訓令式の2つが主流となっていますが, 統一されていませんし, どちらも, 日本語の音韻あるいは正書法のいずれの観点でも厳密に対応していないため, 不完全で欠陥がある古臭い記法です. 実際, このModのロゴも "Shokuhō" という訓令式でもヘプバーン式でもない, 米英の政府機関によって整備された記法に従っています.

一方で, ゲーム内のテキストはヘプバーン式や訓令式が混在しています. 大量の固有名詞が登場するため, 混同を避けるために音韻・正書法の両方の観点である程度厳密な一致が求められます. そのため, ここではBGNおよびPCGNという米英の政府機関によって整備された日本語のローマ字表記法である, 通称修正ヘプバーン式をさらに一部修正した記法を使用することにしました. 修正ヘプバーン式は, 英語の発音を使用して日本語の発音に近づけるという観点では優れていますが, 一方で日本語のかな表記には対応していません. 例えば「おお」「おう」 の表記の違いを識別できません. 結果的に, 修正ヘプバーン式に基づきつつ, かな表記を特定しやすいよう調整した変則的な記法にしています. つまり, 日本語の発音上は同じですが, おう/おお のどちらも ō と書くのではなく, ō/ou で表記を区別することにしました. 発音の観点では, 修正ヘプバーン式以上に発音を近づける工夫はしていません. 例えばJSL式は単語のアクセントも表記する記法ですが, 今回はアクセントは考えないことにしています.

一方で, 記法の変更点はここだけです. 歴史的仮名遣いの違いとか,「へ」「え」の違いまでローマ字で表現するのは難しいため, 現代語でのかな表記に基づいています. 日本語テキストでは, 固有名詞に限定して旧字体や歴史的仮名遣いを採用しています.

補足: 最近, 文化庁の審議会で議論されているローマ字表記方針の改訂が盛んに報道されています. 最新の第8回の議事録を読んだ限りでは, Hepburn式が国際的にも定着していることと, 日本語特有の母音・撥音・促音の表記の難しさに触れられていますが, 解決策は提示されていません. つまり, 現時点では文化庁内の議論でも, 本Modで取り組んでいる問題に対して完全な答えは出されていません.
